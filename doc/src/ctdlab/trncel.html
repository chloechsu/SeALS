SYNOPSIS

function A = trncel(A,delta)

DESCRIPTION 

 Absolute truncation of factor matrices.  All entries less than delta in
 magnitude are removed.

 Input:
   A = ctd
   delta = truncation tolerance. If delta is empty or 0, do nothing.

 Output:
   A = ctd
