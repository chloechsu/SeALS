trncspmat

PURPOSE

For a sparse matrix, truncate all elements less than delta using a mask.

SYNOPSIS

function A = trncspmat(A,delta)

DESCRIPTION 

 For a sparse matrix, truncate all elements less than delta using a mask.  
 There might be a better way to do this, but this way avoids having 
 explicit indexing.
 Input:
   A = sparse matrix
   delta = cutoff threshold
 Ouput:
   A = truncated input
