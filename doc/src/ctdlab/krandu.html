krandu

PURPOSE

Create random sparse ctd with uniformly distributed elements.

SYNOPSIS

function B=krandu(D,N,R,density)

DESCRIPTION 

 Create random sparse ctd with uniformly distributed elements.

 Input:
   D = number of dimensions
   N = scalar or D-vector of dimension sizes
   R = separtion rank
   density = scalar between 0 and 1, approximate fraction of non-zeros.

 Output:
   B = a random sparse ktensor
