DESCRIPTION 

 HadTensProd performs Hadamand tensor products for a ktensor function F 
 and a ktensor function G. The result is a new ktensor function FG.
 Input:
   F - ktensor function.
   G - ktensor function.
 Output:
   FG - the Hadamand product of F and G.
