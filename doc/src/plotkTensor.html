plotkTensor

PURPOSE

Plot a ktensor using different plotting styles

SYNOPSIS

function [ handleOutput ] = plotkTensor( F, gridT, varargin )

DESCRIPTION 

 Plot a ktensor using different plotting styles

 F: ktensor with 'dim' dimensions
